Welkom, mijn naam is Olaf van der Kaaij

Na bijna twintig jaar horeca, ben ik toe aan een nieuwe uitdaging in een nieuw vakgebied. Zeer bewust heb ik gekozen om mijn werkende leven voort te zetten in de IT, en met Make IT Work van de HvA, heb ik een traject gekozen, om dit zo spoedig mogelijk waar te maken. Nu dit traject zo goed als afgerond is, wil ik de opgedane kennis toepassen in deze passie. Als ik mezelf zou moeten omschrijven, ben ik leergierig, heb ik een sterk analytisch vermogen en logische denkwijze. Eigenschappen die naar mijn oordeel goed passen bij een ontwikkelaar. Als opgeleid JAVA developer, kan ik mijn creativiteit, doorzettingsvermogen kwijt in het schrijven van code. Mijn uiteindelijke doel is om me van back-end developer te ontwikkelen tot Full stack developer.
